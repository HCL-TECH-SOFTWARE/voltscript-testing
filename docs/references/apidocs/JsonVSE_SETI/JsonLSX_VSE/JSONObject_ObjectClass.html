    • Properties Detail

      • ChildCount (Get)

        Property Get ChildCount As Integer-Unsigned
        Number of child elements within this JsonObject.

      • JSONType (Get)

        Property Get JSONType As String
        Type of the JsonObject as a string. Useful for logging. To check the JsonObject type in conditionals use the corresponding is... methods.
        Values are:
        - string
        - number
        - array
        - boolean
        - object

      • Label (Set)

        Property Set Label As String
        Label for the JsonObject, blank for a JsonObject that is an array.

      • ParentObject (Get)

        Property Get ParentObject As JSONObject
        JsonObject that contains this JsonObject, if available.

      • ScalarValue (Set)

        Property Set ScalarValue As Variant
        String, number or boolean value of this JsonObject. Only valid for JsonObjects where isScalar is true. Not valid for arrays or objects.

      • ShortValue (Get)

        Property Get ShortValue As String
        First 16 characters of the toString() value of this JsonObject. Useful for debugging or logging purposes to easly identify the current JsonObject from a larger string of JSON.

      • ValueArray (Get)

        Property Get ValueArray As Variant
        Only valid for a JsonObject that contains an array of scalars. Returns a Variant array of the values.

      • AppendToJSONArray

        Sub AppendToJSONArray(Value As Variant) As Boolean
        For JsonObjects that are arrays, appends a value to the array. For new JsonObjects, it appends the value to the array and defines it as an array.
        idx Argument Data Type Attributes Comments
        1ValueVariantValue to add to the array 
         
        Code Sample
        %REM
            The following code appends a string of JSON arrays into an existing JsonObject.
        %END REM
        Dim jp as new JsonParser
        Call jp.loadFromJson(|[]|)
        Dim arrayjo as JsonObject
        Set arrayJo = jp.getrootobject()
        Call arrayJo.appendToJsonArray("Hello")
        Call arrayJo.appendToJsonArray("New")
        Call arrayJo.appendToJsonArray("World")
        Print arrayJo.toString(false)

      • FindObjectByPath

        Function FindObjectByPath(Labels As String) As JSONObject
        Finds a JsonObject within the current JsonObject using an array of labels to navigate down the hierarchy.
        idx Argument Data Type Attributes Comments
        1LabelsStringArrayArray of labels to use to find the JsonObject 
         
        Code Sample
        %REM
            The following example loads a string of JSON and finds the element at "address/addr1"
        %END REM
        Dim parser as New JsonParser()
        Dim jsonObj as JsonObject
        Dim childObj as JsonObject
        Dim json as String
        Dim path(1) as String
        json = |{"address":{"add1":"10 Main Street","add2":"","country":"United Kingdom","county":"","postcode":"NE1 1TT","town":"London"},"age":42,"children":[{"age":12,"firstName":"Jane"},{"age":10,"firstName":"Fred"}],"firstName":"John","lastName":"Doe","living":true}|
        Call parser.loadFromJson(json)
        Set jsonObj = parser.getRootObject()
        path(0) = "address"
        path(1) = "add1"
        Set childObj = jsonObj.findObjectByPath(path)
        Print childObj.toString(True)

      • GetChildren

        Function GetChildren() As JSONObject
        Gets all the children of the current JsonObject, as an array of JsonObjects. Only valid is the JsonObject is an object or an array.
         
        Code Sample
        %REM
            The following code loads a string of JSON and prints the type of all direct children.
        %END REM
        Dim parser as New JsonParser()
        Dim jsonObj as JsonObject
        Dim json as String
        json = |{"address":{"add1":"10 Main Street","add2":"","country":"United Kingdom","county":"","postcode":"NE1 1TT","town":"London"},"age":42,"children":[{"age":12,"firstName":"Jane"},{"age":10,"firstName":"Fred"}],"firstName":"John","lastName":"Doe","living":true}|
        Call parser.loadFromJson(json)
        Set jsonObj = parser.getRootObject()
        ForAll child in jsonObj.getChildren()
           Print child.jsonType
        End ForAll

      • InsertObject

        Sub InsertObject(Object As JSONObject, Label As String) As JSONObject
        Inserts another JsonObject into an existing JsonObject, with an optional label. If no label is provided, the label of the JsonObject is used.
